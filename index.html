    Help & Tips

    Quick answers for using this tracker effectively. Your data never leaves your browser unless you choose to export it.

    Getting started

    Pick a question in the grid, set its status, add notes and an owner. Scoring updates live. The grid is organized by workload (rows) and deployment lane (First / Then / Later columns).

    Save & restore your progress

    Export JSON — the fidelity format. Use this to save a snapshot you'll re-import later, share with another teammate using the app, or commit to source control for review.

    Export CSV — the interop format. Use this to import into Microsoft Planner, review in Excel, or hand to a non-technical stakeholder.

    Import accepts either format. It auto-detects by file extension.

    ⚠️ If you edit the CSV in Excel and save, be careful not to remove or rename the Question ID column — it's required for import to map rows back to questions.

    Move work into Microsoft Planner

    1. Click Export CSV.
    2. Planner Premium / Project Plan 3 customers: In Planner, choose New plan → Import from Excel and upload the file. Buckets, priorities, progress, labels, and notes populate from the CSV.
    3. Planner Basic customers: Native CSV import is not available. Open the CSV in Excel as a working list, or create buckets manually using the Bucket Name column as a guide. The PowerShell script in this repo can also create the Planner tasks via Microsoft Graph if you have admin access.

    Column mapping:

    • Task Name ← question prompt
    • Bucket Name ← workload (Identity, Apps, Network, …)
    • Priority ← criticality (high → Urgent, medium → Medium, low → Low)
    • Progress ← status (mapped to Planner's 3-state model)
    • Assigned To ← owner (see next section)
    • Labels ← deployment lane (First / Then / Later)
    • Notes ← remediation hint + reference link + your notes

    Owner names & Planner assignment

    Planner can only auto-assign tasks when the Assigned To value matches an Entra display name or email in your tenant. For best results, enter owners as their full display name (e.g., Jane Doe) or UPN (jane.doe@contoso.com).

    If a name doesn't match, Planner leaves the task unassigned — no error, but you'll need to assign it manually.

    How scoring works

    Scoring is progress-aware: items marked Completed contribute fully, In Progress partially, Not Started not at all. Not Applicable is excluded from the denominator.

    Top Recommended Actions are ranked by criticality first, then deployment lane, then status maturity, then weight.

    Privacy

    Nothing leaves your browser. There is no server-side storage, no telemetry, and no account required. Snapshots are local downloads only.
